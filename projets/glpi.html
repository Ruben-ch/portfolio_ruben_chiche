Projet GLPI

Gestion de Parc & OCS Inventory - BTS SIO (Cadre Éducatif)

Debian 11 Apache2 MySQL PHP

Contexte Pédagogique

Dans le cadre de ma formation BTS SIO, j'ai réalisé ce projet pour simuler la gestion informatique d'une PME. L'objectif était de coupler GLPI (gestion administrative) avec OCS Inventory (remontée technique) pour obtenir une vision complète du parc.

Mission : Installer et interconnecter ces deux solutions Open Source pour automatiser l'inventaire matériel et gérer les incidents utilisateurs.

Détails techniques

Serveur LAMP

Installation d'une machine virtuelle Debian hébergeant le serveur web Apache et la base de données MySQL.

Liaison LDAP

Configuration de l'annuaire Active Directory pour permettre aux utilisateurs de se connecter avec leurs identifiants de session.

OCS Inventory

Installation du serveur OCS et déploiement de l'agent sur les postes clients pour synchroniser l'inventaire dans GLPI.

Helpdesk

Création de tickets de test, attribution aux techniciens et définition de SLA (délais d'intervention).

Bilan du projet

Ce TP m'a permis de comprendre le fonctionnement d'une solution ITSM complète. J'ai acquis des compétences en déploiement d'agents et en gestion de base de données, essentielles pour un administrateur réseau.
