Windows Server

Active Directory DS & Intégration LDAP/GLPI

Windows Server AD DS DNS / DHCP LDAP

Contexte Pédagogique

Au cœur de toute infrastructure réseau d'entreprise se trouve la gestion des identités. Ce projet BTS SIO visait à déployer un contrôleur de domaine Active Directory (AD DS) pour centraliser l'authentification et sécuriser l'accès aux ressources.

Objectif clé : Interconnecter cet annuaire Microsoft avec des solutions tierces comme GLPI via le protocole LDAP, permettant ainsi une authentification unique (SSO) pour les utilisateurs du support.

Architecture & Services

Contrôleur de Domaine

Installation et promotion de Windows Server. Configuration de la forêt et des zones DNS/DHCP.

Gestion Utilisateurs

Création d'une structure d'Unités d'Organisation (OU), gestion des groupes de sécurité et scripts de démarrage.

GPO (Stratégies)

Déploiement de stratégies de groupe : montage de lecteurs réseau, restriction du panneau de configuration, fonds d'écran.

Liaison LDAP

Configuration du connecteur LDAP dans GLPI pour synchroniser les comptes AD et authentifier les techniciens.

Compétences Acquises

Ce projet m'a permis de maîtriser l'administration système Windows, standard incontournable en entreprise. J'ai appris à garantir l'intégrité des données utilisateurs et à assurer l'interopérabilité entre systèmes hétérogènes (Windows/Linux via LDAP).
